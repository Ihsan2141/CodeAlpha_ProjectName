# CodeAlpha - Secure Coding Review

## Objective

The objective of this project is to review a Python application for security vulnerabilities and recommend secure coding practices.

## Tools Used

- Python
- Manual Code Review
- Kali Linux

## Project Files

- vulnerable_app.py
- Findings.md
- Recommendations.md
- Secure_Code_Review_Report.md

## Outcome

Identified security weaknesses and proposed remediation techniques to improve application security.
