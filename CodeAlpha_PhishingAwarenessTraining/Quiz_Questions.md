# Phishing Awareness Quiz

## Question 1
What is phishing?

A) A network protocol
B) A cyber attack used to steal sensitive information
C) A firewall tool
D) An operating system

Answer: B

---

## Question 2
Which is a common sign of a phishing email?

A) Correct spelling and grammar
B) Trusted sender verification
C) Urgent requests for personal information
D) Official company domain

Answer: C

---

## Question 3
What should you do before clicking a link?

A) Click immediately
B) Check the URL carefully
C) Ignore security warnings
D) Disable antivirus

Answer: B

---

## Question 4
What is social engineering?

A) Building software
B) Manipulating people into revealing information
C) Network configuration
D) Database administration

Answer: B

---

## Question 5
Which practice helps prevent phishing attacks?

A) Reusing passwords
B) Ignoring updates
C) Multi-factor authentication
D) Sharing credentials

Answer: C
