# CodeAlpha - Phishing Awareness Training

## Objective

The objective of this project is to educate users about phishing attacks, social engineering techniques, and online security threats.

## Topics Covered

- Introduction to Phishing
- Types of Phishing Attacks
- Fake Websites and Emails
- Social Engineering Techniques
- Best Security Practices
- Real-World Examples
- Interactive Quiz

## Tools Used

- Kali Linux
- Web Browser
- Microsoft PowerPoint / Google Slides
