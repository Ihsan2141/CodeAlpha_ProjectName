# CodeAlpha Basic Network Sniffer

## Objective
Build a Python-based network packet sniffer to capture and analyze network traffic.

## Features
- Captures network packets
- Displays Source IP
- Displays Destination IP
- Identifies Protocols
- Shows Packet Payload Information

## Tools Used
- Python 3
- Scapy
- Kali Linux

## Learning Outcomes
- Understanding packet sniffing
- Network traffic analysis
- Protocol inspection
- Cybersecurity fundamentals
